# Arrays

## What is an Array?

An array is a collection of elements stored together.

Example:

[4, 7, 2, 9, 1]

## Common Operations

- Access -> O(1)
- Search -> O(n)
- Insert -> O(n)
- Delete -> O(n)

## Key Learnings

- Traversing arrays with loops
- Finding minimum and maximum values
- Counting elements based on conditions
- Calculating sums
