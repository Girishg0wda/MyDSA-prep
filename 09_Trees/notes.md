# Trees

A tree is a hierarchical data structure.

Terminology:

* Root
* Parent
* Child
* Leaf

Binary Tree:
Each node has at most two children.

Example:

```
  10
 /  \
5   15
```

Traversals:

* Preorder (Root Left Right)
* Inorder (Left Root Right)
* Postorder (Left Right Root)

Applications:

* File Systems
* Search Engines
* Databases
* Binary Search Trees
