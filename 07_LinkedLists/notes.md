# Linked Lists

A Linked List is a collection of nodes.

Each node contains:
- Data
- Pointer to next node

Structure:

10 -> 20 -> 30 -> None

Advantages:
- Dynamic size
- Easy insertion and deletion

Disadvantages:
- No direct indexing
- Extra memory for pointers

Common Interview Problems:
- Reverse Linked List
- Detect Cycle
- Find Middle Node
- Merge Lists
